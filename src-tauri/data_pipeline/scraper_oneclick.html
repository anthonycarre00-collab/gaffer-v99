Gaffer — Player Data Scraper

3 steps. No installation. No Python. No terminal. Just paste and click.

1 Open FBref and copy the table

Click this link to open the FBref Big 5 stats page, then select all the text and copy it:

📊 Open FBref Big 5 Stats →

How to copy: On the FBref page, press Ctrl+A (select all) then Ctrl+C (copy).
Or: click anywhere on the stats table, press Ctrl+A twice (selects table first, then whole page), then Ctrl+C.
Don't worry about formatting — this tool handles TSV, CSV, HTML, and plain text.

3 Download the player data file

Click the button. A file called gaffer_players.json will download automatically.

⬇ Click here if download didn't start automatically
What happens next: Once you have gaffer_players.json, it goes into the game's data pipeline folder. The game's world builder converts it into the bundled world database. You only need to do this once — the data ships with the game forever after.

Having trouble? If the paste doesn't parse, try going to FBref → click the "Share & Export" button above the stats table → "Get table as CSV (for Excel)" → copy that CSV text → paste here. That format always works.
